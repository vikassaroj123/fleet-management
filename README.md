# Fleet Focus - Fleet Management System

A comprehensive fleet management system for vehicle maintenance, job cards, inventory tracking, and service scheduling.

## Features

- **Vehicle Management**: Track and manage your entire fleet
- **Job Cards**: Create and manage service job cards
- **Inventory Tracking**: Monitor spare parts and inventory levels
- **Service Scheduling**: Schedule and track vehicle services
- **Document Management**: Manage vehicle documents and certifications
- **Reports & Analytics**: Comprehensive reporting and analytics dashboard
- **Branch Management**: Multi-branch fleet management
- **Driver Management**: Assign and track drivers

## Technologies

This project is built with:

- **Vite** - Fast build tool and dev server
- **TypeScript** - Type-safe JavaScript
- **React** - UI library
- **shadcn-ui** - Beautiful UI components
- **Tailwind CSS** - Utility-first CSS framework
- **React Router** - Client-side routing
- **Recharts** - Data visualization

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

```sh
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Project Structure

```
src/
├── components/     # Reusable UI components
├── context/       # React context providers
├── data/          # Mock data and types
├── hooks/         # Custom React hooks
├── lib/           # Utility functions
├── pages/         # Page components
└── main.tsx       # Application entry point
```

## Development

The development server will start on `http://localhost:5173` by default.
